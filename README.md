# Sales-Analise Using python
This project is a Sales Analyzer built using Python. It reads and processes sales data from CSV files, performs data analysis, and generates meaningful business insights. I have learnt this project from the Youtube : DataSteroid


Features --------
Calculate total revenue and total orders
Track total quantity sold
Analyze revenue by category and product
Identify best-selling products
Find top customers by revenue
Generate daily revenue reports
Calculate revenue within a custom date range
Export category-specific data to separate CSV files


Technologies Used --------
Python
File Handling (CSV)
Dictionaries & Lists
DateTime Module
Data Processing & Analysis

A beginner-friendly project showcasing Python fundamentals, data analysis, and business reporting skills.
